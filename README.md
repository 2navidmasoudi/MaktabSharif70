### Change apache root

You can relocate it by editing the **DocumentRoot** setting in _XAMPP\apache\conf\httpd.conf_ .

It should currently be:

> C:/xampp/htdocs

Change it to:

> C:/Users/PATH/TO/YOUR/DIRECTORY
